## Mihomo 配置

> Windows tun 模式配置文件。[Box for Magisk](https://github.com/taamarin/box_for_magisk) 配置文件。
>
> 自用，仅供参考。Release 是完整配置文件，仅做备份使用。

### 信息

- Windows 内核使用 `amd64-v3` 版本。
- Android 内核使用 `arm64-v8` 版本。
- 基于[官方推荐配置](https://wiki.metacubex.one/example/conf/#__tabbed_1_2)修改。
- 白名单规则，使用[ Mihomo 官方规则集](https://github.com/MetaCubeX/meta-rules-dat)。
- WEB UI 使用 [Zashboard](https://github.com/Zephyruso/zashboard) 无字体版本。
- 分组[图标](https://github.com/ewigl/licons)来源于网络。
- 参考：[官方文档](https://wiki.metacubex.one/config/)。
